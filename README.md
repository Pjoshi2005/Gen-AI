# Custom Tokenizer CLI Tool

A simple Node.js command-line tool that builds a custom tokenizer from input text, supports encoding text into token IDs, decoding token IDs back to text, and handles special tokens like `<SOS>`, `<EOS>`, and `<UNK>`.

---

## Features

- Learns vocabulary dynamically from input text
- Encodes text into token IDs
- Decodes token IDs back to text
- Handles special tokens for start, end, and unknown characters
- Simple interactive command-line interface
